     INTRODUCTION

          The program MDMTST32.EXE is a 32-bit console
          program that simplifies the installation of the
          PC-MDM-1.  This program is described below.

          This disk also contains Drivers for one or more
          languages.  Each language is contained in a
          separate subdirectory on this disk.  Refer to the
          README.TXT file in each subdirectory for
          information on each Driver.



     MDMTST32.EXE

          MDMTST32.EXE performs diagnostic tests on the
          PC-MDM-1.  It verifies that the PC-MDM-1 is
          properly installed and does not conflict with other
          cards.  The installation procedure is described in
          Section 2.0 of the PC-MDM-1 User's Manual.  To run
          the diagnostic tests, enter the following command
          line:

               MDMTST32
