     INTRODUCTION

          The PM429-2 is a PC/104+ card that allows the
          user to communicate with ARINC 429 and/or ARINC 717
          databuses.  The folders listed above contain the drivers,
          libraries, and examples for using the PM429-2 product
          in a user application.

          Please perform the following steps.

          1.  Go to the <DRIVERS> folder to select and
              install the driver files appropriate for your
              operating system.

          2   Go to the <TEST> folder to test the
              installation of the PM429-2 in your computer.

          3.  Go to the <API> folder to copy the headers and
              libraries for writing applications, and the
              example programs that exercise the various
              features of the PM429-2.
