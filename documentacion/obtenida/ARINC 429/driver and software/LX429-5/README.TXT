     INTRODUCTION

          The LP429-5 is a PCI plug-in card and the LE429-5
          is a PCI Express plug-in card to communicate with
          MIL-STD-429 databuses.  The folders listed above
          contain the drivers, libraries, and examples for
          using the LP429-5 and LE429-5 card in a user
          application.

          Please perform the following steps.

          1.  Go to the <DRIVERS> folder to select and
              install the driver files appropriate for your
              operating system.

          2.  Go to the <TEST> folder and test the
              installation of the LP4295 or LE429-5 in your
              computer.

          3.  Go to the <API> folder to copy the headers and
              libraries for writing applications to operate
              the LP4295 or LE429-5, and the example programs
              that exercise the various features of the
              LP4295 or LE429-5.
