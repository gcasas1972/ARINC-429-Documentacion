     INTRODUCTION

          The OmniBus line of interface products allow the
          user to communicate with multiple avionics
          databuses.  These avionics databus interfaces
          include MIL-STD-1553, ARINC 429, ARINC 717, ARINC
          708, RS-422, CSDB, and Discrete IO.  The folders
          listed above contain the drivers for installing an
          OmniBus product under various operating systems.

          Please refer to the README.TXT file for your
          particular OmniBus product for installation
          instructions for that product.
