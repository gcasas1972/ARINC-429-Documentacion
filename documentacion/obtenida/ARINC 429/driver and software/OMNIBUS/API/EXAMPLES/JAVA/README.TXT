     INTRODUCTION

          The folders contain a Java library and examples that operate
          Ballard hardware using the BTIDriver API.  To run the examples,
          the Java Runtime Environment and the Ballard hardware
          drivers must be installed.  Refer to the README.TXT in the
          subfolders for more information.
