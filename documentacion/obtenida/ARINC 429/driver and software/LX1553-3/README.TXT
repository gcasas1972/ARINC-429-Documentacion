     INTRODUCTION

          The LP1553-3 is a PCI card, and the LC1553-3 is a
          cPCI card that allow the user to communicate on a
          MIL-STD-1553 databus.  The folders listed above
          contain the drivers, libraries, and examples for
          using the LP1553-3 and LC1553-3 from a user
          application.

          Please perform the following steps.

          1.  Go to the <DRIVERS> folder to select and
          install the driver files appropriate for your
          operating system.

          2.  Go to the <TEST> folder to test the
          installation of the LP1553-3 or LC1553-3 in your
          computer.

          3.  Go to the <API> folder to unpack the headers
          and libraries for writing applications to operate
          the LP1553-3 or LC1553-3, and the example programs
          that exercise the various features of the LP1553-3
          and LC1553-3.
