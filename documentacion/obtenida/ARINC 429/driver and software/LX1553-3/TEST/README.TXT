     INTRODUCTION

          The program L13TST32.EXE is a 32-bit console
          program that simplifies the installation of the
          LP1553-3 and LC1553-3.



     L13TST32.EXE

          L13TST32.EXE performs diagnostic tests on the
          LP1553-3 and LC1553-3.  It verifies that the L13 is
          properly installed and does not conflict with other
          cards.  The installation procedure is described in
          Section 2.0 of the L13 User's Manual.  To run
          the diagnostic tests, enter the following command
          line:

               L13TST32
