     INTRODUCTION

          The folders listed above contain the driver files
          required by various operating systems to install
          the CM429-1 in your computer.  Refer to the
          README.TXT in the folder appropriate for your
          operating system.

          Once the driver files are installed, refer to the
          <TEST> folder on how to test the installation of
          the CM429-1.
