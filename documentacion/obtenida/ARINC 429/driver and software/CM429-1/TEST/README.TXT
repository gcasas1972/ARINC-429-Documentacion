     INTRODUCTION

          The program C41TST32.EXE is a 32-bit console
          program that simplifies the installation of the
          CM429-1.  This program is described below.



     C41TST32.EXE

          C41TST32.EXE performs diagnostic tests on the
          CM429-1.  It verifies that the CM429-1 is properly
          installed and does not conflict with other cards.
          To run the diagnostic tests, enter the following
          command line:

               C41TST32
