     INTRODUCTION

          The folders listed above contain various examples
          illustrating the operation of the different
          protocols available on BTIDriver-compliant devices.
          Not all protocols may be available on your
          particular hardware.  Refer to the README.TXT in
          each folder for information about each of the
          examples.
