     INTRODUCTION

          There are four examples provided both in C source
          code files and 16-bit DOS executable files.  Each
          of the examples use the utility functions in PM11.H
          and PM11.C.

          Refer to the comments in each example source file
          for an explanation of the code.

          FILE         TITLE
          --------------------------------------------------
          EXAMP1.C     BC mode example
          EXAMP2.C     RT mode example
          EXAMP3.C     Monitor mode example
          EXAMP4.C     Monitor mode with initilization example
