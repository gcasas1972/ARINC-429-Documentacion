     INTRODUCTION

          The program C13TST32.EXE is a 32-bit console
          program that simplifies the installation of the
          CM1553-3.



     C13TST32.EXE

          C13TST32.EXE performs diagnostic tests on the
          CM1553-3.  It verifies that the CM1553-3 is
          properly installed and does not conflict with other
          cards.  To run the diagnostic tests, enter the
          following command line:

               C13TST32
