     INTRODUCTION

          The program P42TST32.EXE is a 32-bit console
          program that simplifies the installation of the
          PC429-1/2.  This program is described below.



     P42TST32.EXE

          P42TST32.EXE performs diagnostic tests on the
          PC429-1/2.  It verifies that the PC429-1/2 is
          properly installed and does not conflict with other
          cards.  To run the diagnostic tests, enter the
          following command line:

               P42TST32
