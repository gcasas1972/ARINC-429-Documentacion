     INTRODUCTION

          The PC-SSB-1 is a printed circuit board that plugs
          into the expansion bus on an IBM-PC/XT/AT or
          compatible computer.  It provides an interface to
          the Space Shuttle Bus.

          The <WIN95> folder contains the driver for the
          PC-SSB-1 operating under Windows 95 and Windows 98.
          For the installation procedure, see the file
          README.TXT in the <WIN95> folder.

          The <WINNT40> folder contains the driver for the
          PC-SSB-1 operating under Windows NT 4.0.  For the
          installation procedure, see the file README.TXT in
          the <WIN95> folder.
