     INTRODUCTION

          The program L43TST32.EXE is a 32-bit console
          program that simplifies the installation of the
          LP429-3 and LC429-3.



     L43TST32.EXE

          L43TST32.EXE performs diagnostic tests on the
          LP429-3 and LC429-3.  It verifies that the L43 is
          properly installed and does not conflict with other
          cards.  The installation procedure is described in
          Section 2.0 of the L43 User's Manual.  To run the
          diagnostic tests, enter the following command line:

               L43TST32
