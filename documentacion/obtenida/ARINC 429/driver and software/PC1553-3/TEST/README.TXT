     INTRODUCTION

          The program P13TST32.EXE is a 32-bit console
          program that simplifies the installation of the
          PC1553-3.  This program is described below.



     P13TST32.EXE

          P13TST32.EXE performs diagnostic tests on the
          PC1553-3.  It verifies that the PC1553-3 is
          properly installed and does not conflict with other
          cards.  To run the diagnostic tests, enter the
          following command line:

               P13TST32
