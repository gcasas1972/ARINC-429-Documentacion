     INTRODUCTION

          The program AX1TST32.EXE is a 32-bit console
          program that simplifies the installation of the
          LP-AFDX-1, LC-AFDX-1, and LM-AFDX-1.



     AX1TST32.EXE

          AX1TST32.EXE performs diagnostic tests on the
          LP-AFDX-1, LC-AFDX-1, and LM-AFDX-1.  It verifies
          that the card is properly installed and does not
          conflict with other cards.  The installation
          procedure is described in Section 2.0 of the
          Lx-AFDX-1 User's Manual.  To run the diagnostic
          tests, enter the following command line:

               AX1TST32
