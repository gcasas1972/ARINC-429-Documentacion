AirDynamicsSim

Sample project that uses script to control autopilot inputs of an Aircraft View.
Timer events adjust the heading hold and altitude hold values (targets). The current
altitude, longitude, lattitude, roll, pitch, etc. respond accordingly.

Note: CoPilot 5.30 or later is required.
