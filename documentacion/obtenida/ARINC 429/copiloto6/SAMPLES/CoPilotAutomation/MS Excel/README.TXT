     INTRODUCTION

          The the file OpenCoPilotFromExcel.xls is a microsoft
          excel spreadsheet developed by Ballard Technology Inc. 
          to demonstrate using CoPilot as an Automation server. 
          This example program is intended to demonstate the following
          CoPilot Automation concepts/tasks:
            - Creating an instance of the CoPilot Application
            - Opening and closing projects
            - Browsing projects for content



     CORRECTIONS, WARNINGS, AND ADDITIONAL INFORMATION

          This example was updated and tested with CoPilot version 5.15.
