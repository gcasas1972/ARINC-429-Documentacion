     INTRODUCTION

          The program VB AUTOMATION FROM example is a 32-bit program 
          developed by Ballard Technology Inc. to demonstrate
          using CoPilot as an Automation server. This example
          program is intended to demonstate the following
          CoPilot Automation concepts/tasks:
            - Creating an instance of the CoPilot Application
            - Opening, closing, running and stopping projects
            - Browsing projects for content
            - Reading and formating monitored data



     CORRECTIONS, WARNINGS, AND ADDITIONAL INFORMATION

          This example was updated and tested with CoPilot version 5.15.
