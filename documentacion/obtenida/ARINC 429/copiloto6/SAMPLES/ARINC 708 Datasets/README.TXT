     This folder contains the following script examples:

      Sample Name          Description
     --------------------------------------------------------------------
     DataAcptToggle.DAT    A sort-of solid color test where the data
                           accept field is toggled every 5 messages. 
                           Data accept 1 has a Range of 80 and data
                           accept 2 has a Range of 40, without filtering
                           on data accept fields you should only see the
                           Range values update on the Radar.  Sends a
                           non-055 labeled message every 90 degrees.

     Flash.DAT             A solid color test rotating Data-Accept and
                           flashing on the faults

     Round Robin.DAT       A solid color test with each fault/warning
                           light being light individualy

     Solid Colors 180.DAT  A solid color test with a scan range of +90
                           to -90

     Solid Colors 360.DAT  A solid color test with a scan range of 0 to
                           360

     Test Pattern.DAT      Rainbow test pattern




     INTRODUCTION

          This folder contains sample datasets for use with
          CoPilot version 3.3.

          To use a sample datasets:

          1) Create a CoPilot project with an ARINC 708 card
             in the hardware tree.

          2) Right-click the 708 channel/board to get its
             context menu.

          3) Select 'Import Dataset' and select one of the
             sample datasets provided.

          4) Open the dataset context menu by right-clicking
             it and select 'View'.
